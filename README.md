# Gaffer

## Vision
An AI matchday butler that helps football fans navigate, discover services, avoid crowds, and enjoy a stress-free stadium experience.

## Target User
Football fans.

## Core Features
- AI Stadium Assistant
- Smart Navigation
- Food Recommendations
- Crowd Awareness
- Matchday Information

## Tech Stack
- React
- Express
- Gemini
- Vite
